# Java-shareit
Сервис аренды вещей - приложение для совместного использования вещей: их можно арендовать, бронировать, оставлять отзывы.

### Приложение дает следующие возможности:
- предлагать вещи для аренды;
- находить нужную вещь и арендовать её;
- оставлять запрос на бронирование;
- бронировать вещь на определённые даты;
- оставлять отзыв о вещи и ее владельце.

### Стек

```PostgreSQL``` ```H2```
